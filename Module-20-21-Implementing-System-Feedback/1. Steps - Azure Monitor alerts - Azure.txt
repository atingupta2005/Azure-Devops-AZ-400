Do send alerts to Azure Function

1. Create Function App - alertfunction, Runtime stack - Powershell Core
2. Open Function App and create a new Function, Template - HTTP Trigger. Function name - alert
3. Open Code+Test and put below code:
  using namespace System.Net
  param($Request, $TriggerMetadata)
  $Request | ConvertTo-Json | Write-Output
4. Monitor\Create New Alert
5. Select Resource -> Virtual Machine
6. Select Condition -> Network Out Total
7. Chose Greater Than Total 1 KB
8. Create Action Group -> Action Type -> Azure Function
9. Enable Common Alert Schema
10. After 10 min check the alert in Monitor
11. Open Azure Function Logs and review the data
