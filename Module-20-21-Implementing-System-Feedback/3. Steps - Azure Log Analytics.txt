Can query data collected in Log Analytics Workspace

1. Create a Log Analytics Workspace
2. Open Advanced Settings\Data\Windows Event Logs
3. Search System and chose it
4. Save
5. Open Workspace Data Sources\Virtual Machines
6. Connect to a couple of VMs
7. Wait for 1 hour to take some data.
8. Open Logs section\Get Started
9. Run Query. Refer to queries.txt
