1. Create ASP.Net Web Application(.Net Framework) with C# language
2. Project name - insights
3. Chose MVC
4. Right Click Project\Add\Application Insights Telemetory
5. Get Started\Or just add the SDK to the local only mode
6. Wait few minutes
7. Run project locally - >IIS Express (Google Chrome)
8. Open Visual Studio\View\Other Windows\Application Insights Search
9. Stop App
10. Create a new Web App in Azure - demoweb20001
11. Runtime Stack -> ASP.Net v4.7
12. Chose Standard App Service Plan
13. In Monitoring Tab\Enbale Application Insights
14. Review and Create
15. Open All Resource\Insights\insight
16. Public Project to newly created Web App
17. Open URL of Web App and click on different links to generate some insights data
18. Have a look at Live Metric of Application insight resource we created
19. Review the tabs Performance, Failures, Users, Sessions
