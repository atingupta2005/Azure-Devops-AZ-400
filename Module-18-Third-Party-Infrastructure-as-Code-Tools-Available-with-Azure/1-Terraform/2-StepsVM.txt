1. Download Terraform or use Azure Cloud Shell
2. Get Subscription ID from the resource
3. Get Tenant ID from Azure AD
4. Update Subscription ID and Tenant ID in the .tf files and also the storage account details
5. Upload 2-machine.tf to Cloud Shell
6. run terraform commands as specified in Terraform Commands.txt
7. Inspect if VM is created
8. Do minor changes in tf file and again apply it.
9. Notice the changes are done in azure portal resource.
