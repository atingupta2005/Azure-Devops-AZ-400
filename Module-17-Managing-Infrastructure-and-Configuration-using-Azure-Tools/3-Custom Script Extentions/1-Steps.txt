Custom Script Extentions is used for ensuring IIS is installed via script on first server boot up

1. Create a new VM and Allow Port 80
2. Create a file named IIS.ps1 with the following contents:
import-module servermanager
add-windowsfeature web-server -includeallsubfeature
3. Create a storage account and create a container in it
4. Upload IIS.ps1 file in Storage Account\Container
5. Open existing VM in Azure Portal
6. Open Extentions\Click Add\Custom Script Extention
7. Attach file from storage account
8. After 10 min inpect custom script extention status and open the URL in browser
