To use credentials from Key Vault and not to store password in ARM template

1. Create a key vault in EastUS named demovault1001
   - Allow "Azure Resource Manager for Template Deployment"
2. Create a secret - vmpassword
3. Grab resource ID of key vault from properties
4. Change parameter template to refer password from keyvault
5. Open Azure Cloud Shell and upload the files - main_kv.json, parameters_kv.json
6. az deployment group create --resource-group az400 --template-file main_kv.json --parameters parameters_kv.json
5. Wait for 5 min and inspect if VM is created
