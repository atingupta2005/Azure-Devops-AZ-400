1. Create Dot Net Core Web App in Visual Studio
2. Edit content of index page - "Welcome to docker"
3. Open project folder in command prompt
4. run - dotnet publish
5. cd <project dir>\bin\debug\netcoreapp3.1\publish
6. dir
7. Copy the published folder to linux vm using winscp
8. Create our Docker file in the directory copied to linux vm - refer Dockerfile
9. Build image - docker build -t dotnetapp .
10. Create container - docker run -d -p 80:80 dotnetapp
11. Check the browser for your app
